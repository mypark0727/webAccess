간격 Preset

  • 숫자 1 증가시 1px씩 증가, 최대 100까지 지원 (0 ~ 100px)
    • 20px 부터는 5px씩 증가
  • 전체 MGN / PAD : margin / padding 의 경우 5px 단위로 증가
  • 주요 컴포넌트는 고유 간격을 포함하고 있으므로 가급적 기본 간격대로 사용하되 특수 디자인에는 멀티클래스로 조합 활용

타이포 .tit* /.txt_*

  • 본문 타이틀(.tit*) 기본설정- 두께: 600(Bold), 간격: 0.727em 0.727em (tit1기준 16px), 행간 1, 자간 -0.6px
  • 본문 스타일(.txt_*) 기본설정- 사이즈 16px(xl 사이즈), 두께 400(Medium), 행간 1.75, 자간 0
  • 영문, 숫자, 한글 : 노토 산스 / 기본 적용되어 있으므로 영문 및 숫자에 개별 스타일링할 필요 없음
